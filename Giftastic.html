Giphy Mood
Giphy API

jQuery/Api


Everyone loves a gif that captures exactly how you're feeling.

I created this dynamic webpage which uses the GIPHY API to populate gifs of a user's choice.

I wanted to create a user interface that was easy for the user to navigate where to enter an emotion so I decided to create a navbar that held a searchbar in the top right. I placed the dynamic buttons underneath the searchbar so that when the user added a new emotion it would append to the last button. I used data classes on the gifs to stop and start animations on click. Another feature I added was an alert if the user clicked "submit" without entering in an emotion.

Technology Used:

Javascript, jQuery, Giphy API, Ajax, Bootstrap, GoogleFonts, Responsive Design
