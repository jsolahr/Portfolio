FORREST GUMP
Trivia Game

JavaScript & Timers


The goal of this project was to create a trivia game that shows only one question at a time until the player answers it or their time runs out. When the user selects an answer they are notified if the answer was correct or incorrect and keeps a tally of their score at the end.

I used Bootstrap for my User Interface and incorporated their button-group for my Forrest Gump questions. This created a simple format for the user to select questions from. To build the questions, I set an array of objects and looped through each question with an on'click listener. I set the timer interval to 15 seconds to allow the user time to think through the question by using the setInterval function. However if they didn't answer it fast enough, the computer would skip to the next question.

Technology Used:

Javascript, jQuery, Bootstrap, GoogleFonts, Responsive Design
