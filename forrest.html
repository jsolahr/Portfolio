FORREST GUMP
Trivia Game

JavaScript & Timers

Forrest Gump is one of my all time favorite movies. I can incorporate a movie quote in most every day conversations... and "that's all I have to say about that..." so I was excited to create a trivia game dedicated to my favorite movie!

The goal of this assignment was to create a trivia quiz that shows only one question at a time until the player answers it or their time runs out. If the player selects the correct answer, the user is notified if the answer was correct or incorrect.

I used Bootstrap for my User Interface and incorporated their button-group for my Forrest Gump questions. This created a simple format for the user to select questions from. To build the questions, I set an array of objects and looped through each question with an on'click listener. I set the timer interval to 15 seconds to allow the user time to think through the question by using the setInterval function. However if they didn't answer it fast enough, the computer would move on to the next question object.
