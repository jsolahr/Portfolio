Welcome to bamazon!

This application mimics an Amazon-like storefront using MySQL. This app takes in orders (on the Command Line) from customers and depletes stock from the store's inventory. When typing node bamazon.js, the application will prompt you to select an item from the store. It will then ask you how many quanities you'd like to purchase. The system will then calculate the cost of the items and deduct it from the inventory. If there is no longer inventory of a selected item, the store will let you know! Enjoy!

Technology Used:

node.js, MySQL, npm packages: inquirer, cTable, mysql
